TC_EDGE_001 – System behavior with extremely long input

Module: Edge Cases

Steps:
1. Navigate to any input field in the application
2. Enter an extremely long string (e.g., 10,000 characters)
3. Attempt to submit or process the input

Expected Result:
System handles the input gracefully without crashing or freezing. Validation message displayed if limits are exceeded.

Priority: Medium
Status: Not executed
