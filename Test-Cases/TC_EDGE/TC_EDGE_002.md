TC_EDGE_002 – System behavior with special characters and symbols

Module: Edge Cases

Steps:
1. Navigate to any input field in the application
2. Enter a string containing special characters (e.g., !@#$%^&*()_+{}[]<>?/|\)
3. Attempt to submit or process the input

Expected Result:
System processes the input safely without errors, crashes, or security vulnerabilities. Validation message displayed if characters are not allowed.

Priority: Medium
Status: Not executed
