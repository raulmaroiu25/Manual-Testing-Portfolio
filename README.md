# Manual Testing Portfolio

This repository contains a complete manual testing portfolio, including:

- Test cases for core functionalities (Login, Registration, Search)
- Form validation test cases
- Boundary value analysis
- Negative test cases
- Usability test cases
- Security test cases
- Performance test cases
- API test cases
- Postman environment and collection (optional)

The purpose of this portfolio is to demonstrate practical QA skills, structured documentation, and the ability to design and organize test scenarios for both UI and API testing.
