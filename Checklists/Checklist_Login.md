# Login Functionality – Checklist

## UI Elements
- [ ] Login page loads successfully
- [ ] Email input field is visible
- [ ] Password input field is visible
- [ ] Login button is visible
- [ ] “Forgot password” link is visible
- [ ] Password field masks characters

## Functional Checks
- [ ] User can enter email
- [ ] User can enter password
- [ ] Login button is clickable
- [ ] Login with valid credentials works
- [ ] Login with invalid password shows error
- [ ] Login with invalid email format shows error
- [ ] Login with empty fields shows validation messages

## Validation
- [ ] Email format is validated
- [ ] Password minimum length is validated
- [ ] Error messages are displayed correctly
- [ ] Error messages disappear after correction

## Security
- [ ] Password is not visible in plain text
- [ ] No sensitive data in URL
- [ ] No SQL injection possible in fields
- [ ] Brute force protection (if applicable)

## Performance
- [ ] Login response time < 2 seconds
- [ ] Page loads without delays

## Compatibility
- [ ] Works on Chrome
- [ ] Works on Firefox
- [ ] Works on Edge
- [ ] Works on mobile resolution
